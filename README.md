# toDoList_Python

A Simple Flask Todo App using SQLAlchemy and SQLite database.

### Setup

Create project with virtual environment

```console
$ mkdir myproject
$ cd myproject
$ python -m venv .env
```

Activate it

```console
$ . .env/bin/activate
```

Install Flask

```console
$ pip install Flask
$ pip install Flask-SQLAlchemy
```

Run the app

```console
$ python app
```
